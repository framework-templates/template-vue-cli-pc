# template-vue-cli-pc

> 基于 vue-cli、element-ui 实现的一套开箱即用的 PC 框架

<h2 align="center">已实现</h2>

- jest单元测试
- elemetn按需引入
- 命令打包分析模块
- 开箱即用
- svg-icon

## 使用步骤

```bash
# 安装依赖
npm install

# 使用webpack启动服务
npm run serve

# 使用webpack打包多个环境
npm run build:dev
or
npm run build:uat
or
npm run build:prod

# 使用单元测试
npm run test:unit

# 打包分析
npm run build:analyzer

```
